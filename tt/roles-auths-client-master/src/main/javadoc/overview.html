roles-auths-client library usage example

Please notice that Java 8 or higher is required. 
    Properties props = new Properties();
    props.load(new FileInputStream("application.properties"));

    XRoadClientConfig config = new XRoadClientConfig(
        new ClientHeaderDetails(props),
        new ServiceHeaderDetails(props),
        new HashSet<>(Arrays.asList(new XRoadServer[] {
                new XRoadServer((String) props.get("xroad_host1")),
                new XRoadServer((String) props.get("xroad_host2"))
        })));

    ClientFactory cf = new RiClientFactory(config);

    HpaClient hc = cf.hpaClient();

    Authorization auth = hc.isAuthorized(userId, delegateId, principalId, null);
HeaderDetails classes contain information about the properties that needs to be set. @see fi.vm.kapa.rova.client.xroad.ClientHeaderDetails ClientHeaderDetails @see fi.vm.kapa.rova.client.xroad.ServiceHeaderDetails ServiceHeaderDetails
